```

Screenshot 3: READ MODE       → Tap & read in seconds
Screenshot 4: AUDIO MODE      → Toggle to listen anywhere

Screenshot 6: HIGHLIGHT       → Save the best lines
Screenshot 7: CATEGORIES      → Endless variety

```

**Rules per screenshot:**
- Title: 3–6 words, bold, benefit-led
- Subtitle: 4–8 words, supporting proof
- 1 idea per screen — no clutter
- Numbers + Power Words + Local Language

## 📋 Pre-Launch Essentials

### Must-Haves Before Submitting:

- [ ] App icon (512×512 PNG)
- [ ] Feature graphic (1024×500 PNG)
- [ ] 8 phone screenshots (1080×1920)
- [ ] Preview video (30 sec, MP4)
- [ ] Privacy policy URL (localized)
- [ ] App tested on 5+ devices
- [ ] Crash rate < 1%
- [ ] App size < 30MB (mass markets)

## ⚡ Speed Tips

1. **Start small** — Launch 1 market first, learn, then scale
2. **Reuse assets** — English assets work for USA + UK + diaspora
3. **Iterate fast** — Update every 2-3 weeks based on data
4. **Native review** — Always have native speakers check translations
5. **Track everything** — You can't improve what you don't measure
